Java中的优先队列

Java中的PriorityQueue默认是小顶堆，逻辑上是堆，物理上是数组，每次取出的元素是最小的。【可以通过比较器的方式传入不同的比较规则】

特点：

  • 实现了Queue接口
  • 不允许放入null元素
  • 插入方法：
    • add()和offer(),前者插入失败抛出异常，后者插入失败返回false。
  • 获取堆顶元素：
    • element()和peek():都是获取堆顶元素，前者获取失败抛出异常，后者返回null
  • 获取并删除队首元素：
    • remove()和poll()都是获取并删除队首元素，前者失败抛出异常，后者失败返回null。
