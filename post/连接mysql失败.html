ERROR 2003 (HY000): Can’t connect to MySQL server on ‘localhost:3306’ (10061)

问题描述：

  • 输入上传命令之后，出现标题上的错误。
  • 数据库无法成功连接

解决方案：

  • 打开win搜索栏，输入“计算机管理”
  • 选择服务
  • 找到mysql，并启动即可
